Recroute

Recroute is the first application project that I developed with the team, this application was made using Kotlin and layouting. In this application project was done by 5 people and I acted as a programmer, this application took up to 3 weeks for almost all of the features to be made. This application is a project that was done during the registration of the Raion organization. This application is an application that I made when I first learned about making android applications, so there are still many shortcomings in the application.

This application itself is an online platform that provides online courses that can be taken by all audiences, especially students. if each person successfully completes the course then he will get a certificate that can be used to register for a job directly on the application.

App Features

Providing Employment

This application can help everyone to have the opportunity to compete in the world of work, especially fresh graduate students

This application utilizes firebase as authentication and database storage.

User can change their personal data
