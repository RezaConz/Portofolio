Cleaving

Cleaving is an app-based online platform built by 5 people and I act as a programmer, this app is made with kotlin and layouting. This application helps people to exchange recyclable waste into points that can be exchanged for various things. In this project I have implemented clean code and clean architecture so that everyone can understand it more easily

App Features

eco-friendly

This application helps people to be able to recycle waste in a simpler way.

Utilizing Firebase

This application utilizes firebase as authentication and database storage.

Editable Data

User can change their personal data

Secured

Guaranteed data security from Firebase

Cloud Storage

Data is stored in the firebase cloud so that the data can easily provide flexibility for users
