Analyze Data in Big Data Environment

This project is the final project of the big data and analytics course. this project performs data analysis in a big data environment. The analysis carried out is to determine customer segmentation using Kmeans Clustering. The dataset is stored in HDFS and the analysis process is carried out on Jupyter Notebook. Customer segmentation is done to determine the right sales strategy to several categories of customers
